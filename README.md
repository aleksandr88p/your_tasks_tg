# 🤖 YourTasks Telegram Bot

Telegram-бот для управления задачами и отслеживания времени через YourTasks API.

## 📋 О проекте

YourTasks Telegram Bot - это удобный интерфейс для взаимодействия с YourTasks API, который позволяет пользователям:
- Создавать и управлять задачами
- Логировать потраченное время
- Просматривать статистику
- Получать уведомления о простоях

**Технологический стек:**
- Python 3.12+
- aiogram 3.x - фреймворк для Telegram-ботов
- YourTasks API - REST API для управления задачами

## 🚀 Текущий функционал (MVP)

### Базовые команды
- `/start` - Приветствие и начало работы
- `/help` - Справка по командам
- `/lang` - Смена языка (Русский/English)

### Управление задачами
- `/new <название>` - Создать новую задачу
- `/list` - Показать все задачи пользователя
- `/task <id>` - Показать детали конкретной задачи
- `/complete <id>` - Отметить задачу как выполненную
- `/active <id>` - Активировать задачу

### Работа со временем
- `/time <id> <минуты> [комментарий]` - Добавить время к задаче
- `/log` - Показать все логи времени
- `/stats` - Общая статистика пользователя
- `/task_stats <id>` - Статистика по конкретной задаче

## 🔧 Быстрый старт

1. Клонирование репозитория:
```bash
git clone <repository-url>
cd your_tasks_tg
```

2. Создание виртуального окружения:
```bash
python -m venv venv
source venv/bin/activate  # для Windows: venv\Scripts\activate
```

3. Установка зависимостей:
```bash
pip install -r requirements.txt
```

4. Настройка переменных окружения:
```bash
cp .env.example .env
# Отредактируйте .env файл
```

5. Запуск бота:
```bash
python bot/main.py
```

## 🔑 Переменные окружения

```env
# Telegram Bot Token (получить у @BotFather)
BOT_TOKEN=your_telegram_bot_token

# YourTasks API URL
API_URL=http://185.182.82.203:8000

# Настройки логирования
LOG_LEVEL=INFO

# Язык по умолчанию
DEFAULT_LANGUAGE=ru
```
