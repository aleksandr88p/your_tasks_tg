# 🤖 YourTasks Telegram Bot

Telegram-бот для управления задачами и отслеживания времени через YourTasks API.

## 📋 О проекте

YourTasks Telegram Bot - это удобный интерфейс для взаимодействия с YourTasks API, который позволяет пользователям:
- Создавать и управлять задачами
- Логировать потраченное время
- Просматривать статистику
- Использовать интуитивный интерфейс с кнопками

**Технологический стек:**
- Python 3.12+
- aiogram 3.x - фреймворк для Telegram-ботов
- YourTasks API - REST API для управления задачами
- Supabase - база данных

## ✅ Что реализовано (v2.0)

### 🎯 Пошаговый интерфейс
- **Интерактивное создание задач** - `/new` → ввод названия
- **Выбор задач по номерам** - больше не нужно запоминать большие ID
- **Пошаговое добавление времени** - `/time` → выбор задачи → минуты → комментарий
- **Умное управление статусами** - `/complete` и `/active` с выбором из списка

### � Главное меню с кнопками
- **📝 Новая задача** - пошаговое создание
- **🟢 Активные задачи** - только активные задачи
- **✅ Завершенные задачи** - только завершенные задачи
- **📋 Все задачи** - все задачи вместе
- **⏰ Добавить время** - логирование времени
- **📊 Статистика** - подробная статистика
- **🏁 Завершить задачу** - завершение по выбору
- **🔄 Активировать задачу** - активация по выбору
- **❌ Отмена** - отмена любого действия
- **❓ Помощь** - обновленная справка

### 📋 Полный набор команд
- `/start` - Показать главное меню
- `/new` - Создать новую задачу (пошагово)
- `/list` - Показать все задачи
- `/time` - Добавить время к задаче (пошагово)
- `/complete` - Завершить задачу (выбор из списка)
- `/active` - Активировать задачу (выбор из списка)
- `/stats` - Общая статистика пользователя
- `/help` - Справка по командам

## 🔧 Быстрый старт

1. Клонирование репозитория:
```bash
git clone <repository-url>
cd your_tasks_tg
```

2. Создание виртуального окружения:
```bash
python -m venv venv
source venv/bin/activate  # для Windows: venv\Scripts\activate
```

3. Установка зависимостей:
```bash
pip install -r requirements.txt
```

4. Настройка переменных окружения:
```bash
cp .env.example .env
# Отредактируйте .env файл
```

5. Запуск бота:
```bash
python bot/main.py
```

## 🔑 Переменные окружения
